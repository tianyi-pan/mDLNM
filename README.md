# mDLNM

**mDLNM** is a meta-package for fitting multiple exposure distributed lag non-linear models (DLNMs). 

The package supports the following four types of multiple exposure DLNMs:

1. **Single-index adaptive cumulative exposure DLNM (Single-Index ACE-DLNM)**: [SIaceDLNM](https://github.com/tianyi-pan/SIaceDLNM)
2. **Additive adaptive cumulative exposure DLNM (Additive ACE-DLNM)**: [aceDLNMadditive](https://github.com/tianyi-pan/aceDLNMadditive)
3. **Single-index distributed response function DLNM (Single-Index DRF-DLNM)**: [SIdrfDLNM](https://github.com/tianyi-pan/SIdrfDLNM)
4. **Additive distributed response function DLNM (Additive DRF-DLNM)**: [drfDLNMadditive](https://github.com/tianyi-pan/drfDLNMadditive)


## Installation
+ Install the development version from GitHub:
```R
# install.packages("devtools")
devtools::install_github("tianyi-pan/mDLNM")
```
