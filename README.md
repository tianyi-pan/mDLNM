# mDLNM

**mDLNM** is a meta-package for fitting multiple exposure distributed lag non-linear models (DLNMs). 

The package supports the following four types of multiple exposure DLNMs:

1. **Single-index adaptive cumulative exposure DLNM (Single-Index ACE-DLNM)**
2. **Additive adaptive cumulative exposure DLNM (Additive ACE-DLNM)**
3. **Single-index distributed response function DLNM (Single-Index DRF-DLNM)**
4. **Additive distributed response function DLNM (Additive DRF-DLNM)**


## Installation
+ Install the development version from GitHub:

# install.packages("devtools")
devtools::install_github("tianyi-pan/mDLNM")
